Gene Cloning Visualizer

An interactive full-stack web application that simulates the gene cloning process step-by-step with dynamic animations and visual explanations.

HERE IS THE DEPLOYED VERSION
https://gene-cloning-visualiser-xq7u.vercel.app/

🚀 Project Overview

The Gene Cloning Visualizer is designed to help users understand the biological process of gene cloning through an engaging and interactive interface.

Instead of static theory, this application provides a visual simulation of each stage — from gene mapping to cloning — making complex concepts easier to grasp.

🎯 Features
🧬 Step-by-step gene cloning simulation
🎮 Interactive “Next Step” navigation
🎨 Real-time DNA helix animation using HTML Canvas
✂️ Visual representation of DNA cutting
🔵 Plasmid (vector) visualization
🧫 Host cell transformation animation
🎉 Final cloning simulation
🧠 Gene Cloning Process Covered
Gene Mapping
DNA Extraction & Cutting
Vector (Plasmid) Preparation
Gene Insertion
Transformation into Host Cell
Cloning & Multiplication

⚙️ Installation & Setup
1️⃣ Clone the repository
git clone https://github.com/your-username/gene-cloning-visualizer.git
cd gene-cloning-visualizer
2️⃣ Setup Backend
cd backend
npm install
npm start
3️⃣ Setup Frontend
cd frontend
npm install
npm start
▶️ Usage
Open the app in your browser (http://localhost:3000)
Click “Start Cloning”
Navigate through each step using the Next Step button
Observe the real-time visual simulation
